
# TeamSpeak Verifizierungssystem

Mit dem TeamSpeak Verifizierungssystem von #Game OS, kannst du User dazu auffordern sich auf deinem TeamSpeak Server zu verifizieren. Dazu müssen deine Nutzer nur eine einfache Webseite öffnen

## Funktionen

  - Client Identifizierung
  - Ränge vergabe
  - Sleep Funktion
  - Avatar
  - Erkennung, wer bereits verifiziert ist.
  - Verifizierungssnachricht festlegen

## Installation:

  - Stelle sicher, dass du einen Webserver mit PHP auf deinem Server laufen hast
  - Entpacke die Dateien in einem beliebigen Ordner in deinem Webverzeichnis
  ```sh
$ wget https://github.com/marv-th/TeamSpeak-Verifizierungssystem/archive/master.zip
$ apt install unzip #nur wenn noch nicht installiert
$ unzip master.zip
```
  - Richte den Bot in der config.php ein
  - Der System sollte Einsatzbereit sein
